# Basic Git Commands

## Setting Up the Rep
* git clone <repo ssh link from git/github>, i.e. `git clone git@github.com:CoderAcademy-ALL/shared-notes.git`

## making Changes to Repo
* `git status`
* git add <filename>
* `git commit -m"Commit message"`
* `git push origin master`

## Getting info
* `git log`
